管理ページ - 有楽自治会 防災マップ
自治会境界の描画と、避難所の追加・編集ができます
ようこそ。タブで作業モードを切り替えてください。
使い方: 地図の左上にあるツールバーから 多角形 アイコンをクリック → 地図を順にクリックして頂点を置く → 最後にダブルクリックで確定 → 「境界をダウンロード」で yuraku-boundary.geojson を取得します。
ダウンロードしたファイルを反映する方法 (初心者向け)
  1. ブラウザで GitHubの data フォルダ を開く
  2. 右上の 「Add file」→「Upload files」 をクリック
  3. ダウンロードした yuraku-boundary.geojson をドラッグ＆ドロップ (同名なので上書きされます)
  4. 下の 「Commit changes」 ボタンを押す (コミットメッセージはそのままでOK)
  5. 1〜3分待つと本番サイトに自動反映されます
使い方: 「避難所を追加」ボタンを押してから地図上の避難所の場所をタップ → 名称や住所などを入力して保存 → 全部登録したら「避難所をダウンロード」で shelters.geojson を取得します。既存のマーカーをタップすると編集できます。
ダウンロードしたファイルを反映する方法 (初心者向け)

  2. 右上の 「Add file」→「Upload files」 をクリック

  4. 下の 「Commit changes」 ボタンを押す
